Vhost API - Speech Functions: Text to Speech functions (Scene embedded in HTML)

sayText(txt,voice,lang,engine)


The "Text to Speech" audio may be triggered in any of 3 ways:
  • When the scene is loaded, using the "Event Functions":
          function vh_sceneLoaded(){
                //the scene begins playing, add actions here
                sayText('Welcome to the virtual host text to speech',1,1,3);
          }

  • Using a call to the sayText() command, specifying the text: Click here to say something!
